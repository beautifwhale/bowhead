# Bowhead monorepo

Bowhead is a Create-React-App template for fast MicroSaas prototyping.

## Includes

- ReactJS
- MaterialUI
- Stripe subscription management
- Firestore database
- Netlify hosting and functions

## How to use

```bash
npx create-react-app my-app --template @mortond/cra-template-bowhead
```
